Zabbix 7.4 – Installation et Monitoring Web

Partie 1 : Installation sur Debian 13 (VMWare)

Prérequis

  • sudo apt update && sudo apt upgrade
  • PHP : sudo apt install php php-mysqli
  • MariaDB : sudo apt install mariadb-server puis sudo mariadb-secure-installation
  • Apache : sudo apt install apache2 libapache2-mod-php

Installation Zabbix

wget https://repo.zabbix.com/zabbix/7.4/release/debian/pool/main/z/zabbix-release/zabbix-release_latest_7.4+debian13_all.deb
sudo dpkg -i zabbix-release_latest_7.4+debian13_all.deb
sudo apt update
sudo apt install zabbix-server-mysql zabbix-frontend-php zabbix-apache-conf zabbix-sql-scripts zabbix-agent2

Base de données

create database zabbix character set utf8mb4 collate utf8mb4_bin;
create user zabbix@localhost identified by 'ton_mot_de_passe';
grant all privileges on zabbix.* to zabbix@localhost;
zcat /usr/share/zabbix-sql-scripts/mysql/server.sql.gz | mysql -uzabbix -p zabbix

Configuration & démarrage

  • Éditer /etc/zabbix/zabbix_server.conf → ajouter DBPassword=ton_mot_de_passe
  • sudo systemctl restart zabbix-server zabbix-agent2 apache2
  • Accès : http://IP/zabbix → Admin / zabbix (changer le mot de passe immédiatement !)

Partie 2 : Scénario Web de monitoring

  1. Créer hôte "LDLC" (groupe : Linux servers)
  2. Créer scénario web :
    • Nom : Zabbix doc LDLC
    • Intervalle : 10s – Tentatives : 10
  3. Ajouter étape :
    • URL : https://www.ldlc.com/
    • Chaîne requise : "populaires"
    • Code : 200
  4. Créer déclencheur : Download speed > 5 → Sévérité Haute
  5. Vérifier dans Surveillance → Web
