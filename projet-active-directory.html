Mise en place d’un Active Directory sous Windows Server

Environnement de test sous VirtualBox : serveur AD + poste client, réseau interne isolé.

1. Préparation

  • Adresses IP statiques sur serveur et client.
  • Désactiver IPv6 et pare-feu (test).
  • Tester ping entre machines.
  • Installer VirtualBox Guest Additions.

2. Installation AD DS + DNS

  • Ajouter rôle "Active Directory Domain Services" via Gestionnaire de serveur.
  • Promouvoir en contrôleur de domaine → Nouvelle forêt (ex. : wsrv.labo).
  • Créer compte admin dédié (ex. admin_AD).

3. Utilisateurs et rattachement

  • Créer utilisateur test (ex. BTSSIO / Passw0rd).
  • Rattacher client au domaine → Redémarrer.

4. Structure OU, Groupes, GPO

  • Créer OU société → Sous-OU par service.
  • Groupes : Sécurité / Distribution, étendues (Global, Universel...).
  • GPO : Mappage lecteurs, fond d'écran, redirection dossiers, etc.

5. Profils et scripts

  • Profils itinérants / obligatoires.
  • Scripts de connexion (.bat) dans SYSVOL\Scripts.

6. Bonnes pratiques

  • Politique mots de passe stricte.
  • Désactiver comptes inactifs.
  • Surveiller journaux + gpupdate /force.
