Installation et paramétrage d’un serveur LAMP sous Debian

Objectif : Mettre en place un serveur web LAMP (Linux, Apache, MariaDB, PHP) en environnement virtualisé (VirtualBox), avec configuration réseau et accès distant SSH.

1. Mise à jour du système

  • sudo apt update
  • sudo apt upgrade

Confirmer les mises à jour avec 'O' si demandé.

2. Configuration et vérification du réseau

  • Vérifier la carte réseau : ip a ou sudo apt install net-tools && ifconfig
  • Dans VirtualBox : Mode réseau → Accès par pont (bridge) sur la carte physique (Wi-Fi/Ethernet).
  • Configuration manuelle (si besoin) : Éditer /etc/network/interfaces avec nano, puis redémarrer : /etc/init.d/networking restart ou ifdown enp0s3 && ifup enp0s3.
  • Vérifier la connectivité : ping vers routeur, Internet, etc.

3. Mise en place d’un accès distant SSH

  • Installation : sudo apt install openssh-server
  • Créer un utilisateur dédié : sudo adduser userssh (mot de passe : mdp)
  • Connexion depuis client (ex. MobaXterm) : ssh userssh@IP_DU_SERVEUR
  • Passer root si besoin : su -

4. Conclusion

Le serveur est maintenant mis à jour, accessible à distance et prêt à accueillir des applications web (GLPI, Moodle, WordPress, etc.).
